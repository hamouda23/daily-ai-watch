## 🔒 Cybersécurité IA

- **CVE-2026-33626 — LMDeploy SSRF (CVSS 7.5), exploitée en 12h** — Divulguée le 21 avril, cette vulnérabilité de Server-Side Request Forgery dans LMDeploy a été armée en moins de 12 heures. Les attaquants l'utilisent pour scanner le réseau interne, accéder aux métadonnées cloud et énumérer des services via un endpoint vision-LLM. Mise à jour immédiate recommandée. [The Hacker News](https://thehackernews.com/2026/04/lmdeploy-cve-2026-33626-flaw-exploited.html)

- **CVE-2026-41349 — OpenClaw : contournement de consentement agentique** — Dans OpenClaw < 2026.3.28, les agents LLM peuvent désactiver silencieusement l'approbation d'exécution des actions via le paramètre `config.patch`. Permet des opérations non autorisées sans consentement utilisateur. [IBM X-Force](https://www.ibm.com/think/x-force/agentic-ai-growing-fast-vulnerabilities)

- **CVE-2025-53773 — GitHub Copilot RCE (CVSS 9.6)** — Injection de prompt cachée dans les descriptions de Pull Request permettant l'exécution de code à distance via GitHub Copilot. [Cycode](https://cycode.com/blog/ai-security-vulnerabilities/)
