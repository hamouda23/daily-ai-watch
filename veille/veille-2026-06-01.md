## 🔥 Faits saillants du jour

1. **NVIDIA RTX Spark & l'ère de l'IA agentique sur PC** — Jensen Huang déclare lors du GTC Taipei (aujourd'hui, 1er juin) que « l'IA agentique et utile est arrivée ». NVIDIA annonce le superchip RTX Spark (CPU Grace 20 cœurs + GPU Blackwell + 128 Go LPDDR5X unifié, 300 Go/s de bande passante), disponible en octobre 2026 chez Dell, HP, Lenovo, Microsoft, Asus, MSI. Windows est repositionné comme OS d'agents IA.

2. **Anthropic : Claude Opus 4.8 + levée $65 Mrd à $965 Mrd de valorisation + Mythos imminents** — Anthropic a levé 65 milliards de dollars (29 mai) à une valorisation record de 965 milliards. Claude Opus 4.8 est sorti le 28 mai (+5 pts sur le coding agentic, 69.2%), et les modèles Mythos — capables d'identifier des vulnérabilités logicielles vieilles de décennies — seront disponibles publiquement « dans les semaines à venir » via le Programme Glasswing pour les défenseurs en sécurité critique.

3. **Premier zero-day généré par IA confirmé par Google GTIG** — Le 11 mai 2026, Google Threat Intelligence Group a détecté et neutralisé ce qui est décrit comme le premier exploit zero-day développé avec l'aide d'une IA. Un acteur malveillant a utilisé un modèle LLM pour découvrir et exploiter une vulnérabilité dans un outil d'administration web open source. Moment charnière pour la cybersécurité offensive.

4. **Robinhood lance le trading agentic** — Le 27 mai 2026, Robinhood ouvre le premier compte de trading exécutable par des agents IA tiers (construits sur AutoGen/LangGraph/AG2). Architecture isolée, limites de capital définies par l'utilisateur, kill switch immédiat. Première infrastructure mainstream permettant à des systèmes multi-agents de trader en conditions réelles.

5. **EU AI Act : délai prolongé de 16 mois** — L'accord de trilogue du 7 mai 2026 (Digital Omnibus) repousse la deadline pour les systèmes IA à haut risque (Annexe III) d'août 2026 à **décembre 2027**. Les règles sur la transparence et les modèles GPAI restent en vigueur au 2 août 2026. Répit significatif pour les institutions financières.

## 🔒 Cybersécurité IA

### Incidents & vulnérabilités critiques

**Premier zero-day généré par IA — Google GTIG (11 mai 2026)**
Un acteur de menace a utilisé un LLM pour identifier et exploiter une vulnérabilité dans un outil d'administration web open source, incluant un script Python de contournement du 2FA. Google a coordonné la divulgation responsable avant exploitation de masse. Le modèle Gemini n'était pas impliqué dans l'attaque. Source : [CNBC](https://www.cnbc.com/2026/05/11/google-thwarts-effort-hacker-group-use-ai-mass-exploitation-event.html) | [SecurityWeek](https://www.securityweek.com/google-detects-first-ai-generated-zero-day-exploit/)

**EchoLeak (CVE-2025-32711) — Attaque zero-click Microsoft 365 Copilot**
Un email unique permet l'exfiltration silencieuse de fichiers internes via Microsoft 365 Copilot. 4 contournements chaînés : bypass du classifier XPIA, redirection Markdown masquée, images auto-fetched, proxy Microsoft Teams. CVSS 9.3. Patché mais surface structurelle persistante. Source : [Varonis](https://www.varonis.com/blog/echoleak) | [ArXiv 2509.10540](https://arxiv.org/abs/2509.10540)

**PROMPTSPY / PROMPTFLUX / PROMPTSTEAL — Premier malware Android IA natif**
PROMPTSPY est le premier malware Android intégrant Gemini dans sa boucle d'exécution pour contrôler le device via Accessibility Service. PROMPTFLUX et PROMPTSTEAL génèrent du code malveillant polymorphique via APIs LLM à la demande. Source : [BleepingComputer](https://www.bleepingcomputer.com/news/security/promptspy-is-the-first-known-android-malware-to-use-generative-ai-at-runtime/)

**Vulnérabilité RCE critique dans le cœur de MCP — OX Security**
40+ CVEs dans les implémentations MCP (Python, TypeScript, Java, Rust) depuis janvier 2026. OX Security a découvert une RCE systémique affectant 9/11 marketplaces MCP, permettant l'exécution de commandes arbitraires et l'accès aux bases de données et clés API. Source : [OX Security](https://www.ox.security/blog/the-mother-of-all-ai-supply-chains-critical-systemic-vulnerability-at-the-core-of-the-mcp/) | [VentureBeat](https://venturebeat.com/security/mcp-stdio-flaw-200000-ai-agent-servers-exposed-ox-security-audit)

**TeamPCP (UNC6780) — 4 dépôts IA compromis en 50 jours (mars 2026)**
Supply chain compromise de LiteLLM, Trivy, Checkmarx et BerriAI via paquets PyPI malveillants et PRs injectés, déployant SANDCLOCK pour voler clés AWS, tokens GitHub et secrets cloud. Millions de téléchargements quotidiens affectés.

**CVE-2025-53773 — RCE dans GitHub Copilot via prompt injection (CVSS 9.6)**
Injection dans les descriptions de PR permettant une exécution de code à distance sur les environnements Copilot.
