## 🔒 Cybersécurité IA

### Bilan de la semaine (16–22 juin 2026)

**Prompt injection : 73 % des déploiements production touchés**
Les données compilées en 2026 révèlent que 73 % des déploiements IA en production présentent des vulnérabilités à la prompt injection. OWASP maintient ce vecteur en position #1 de son Top 10 LLM. OpenAI a publiquement qualifié le problème de "frontier security challenge" sans solution propre à ce jour.

**CVE-2025-53773 (CVSS 9.6) — GitHub Copilot : RCE via PR**
Rappel critique : injection cachée dans les descriptions de pull requests → exécution de code arbitraire via Copilot. Score CVSS 9.6. Toutes les équipes utilisant Copilot sur des repos partagés sont exposées.

**Attaques automatisées documentées (Unit 42 / Munich Re)**
Unit 42 (Palo Alto) a documenté en mars 2026 les premières attaques massives d'injection de prompt indirecte en conditions réelles : contournement de filtres publicitaires, fuite de system prompts sur plateformes commerciales. Techniques utilisées : "InfoFlood" (surcharge contextuelle) et "adversarial suffixes" (chaînes déstabilisant les mécanismes de sécurité internes). Munich Re classe la prompt injection comme "major attack vector" dans son rapport cyber 2026, soulignant son coût faible et sa scalabilité pour les adversaires.

**Tendances**
- Les agents codeurs autonomes (Cursor, Copilot, Claude Code) deviennent des vecteurs d'attaque de premier plan.
- Les attaques hybrides (prompt injection + exploits web classiques) sont désormais documentées en production.

**Anthropic sur Colossus 1 (SpaceX/xAI)** : Anthropic a sécurisé l'accès exclusif à la totalité du datacenter Colossus 1 (Tennessee) — 300 MW et plus de 220 000 GPU NVIDIA.
