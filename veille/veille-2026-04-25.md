## 🔒 Cybersécurité IA

### Recherches & publications

- **ArXiv / IEEE S&P 2026 — Injection de prompt dans les plugins tiers** — Première étude à grande échelle portant sur 17 plugins tiers déployés sur plus de 10 000 sites web. Résultat clé : quand les limites de rôle sont violées, les taux de succès d'attaque passent de 0–25 % à 25–100 %. Accepté à l'IEEE Symposium on Security and Privacy 2026. [(ArXiv 2511.05797)](https://arxiv.org/abs/2511.05797)

- **ArXiv — "From Prompt Injections to Protocol Exploits"** — Étude systématique classifiant plus de 30 techniques d'attaque ciblant les agents LLM : manipulation d'input, compromission du modèle, attaques système/vie privée et exploits au niveau des protocoles. Introduit un modèle de menace bout-en-bout unifié pour les écosystèmes d'agents. [(ArXiv 2506.23260)](https://arxiv.org/abs/2506.23260)

- **ArXiv — Injection sur assistants de codage agentiques** — Taxonomie tridimensionnelle des attaques ciblant les coding assistants IA (ex. GitHub Copilot, Cursor) : vecteurs de livraison, modalités d'attaque et comportements de propagation. [(ArXiv 2601.17548)](https://arxiv.org/abs/2601.17548)

### Tendance

Le RAG poisoning s'impose comme vecteur d'attaque critique et sous-estimé : les adversaires injectent des documents malveillants dans les corpus de récupération, compromettant silencieusement les systèmes RAG sans toucher directement le modèle. La combinaison avec les workflows agentiques crée un risque de propagation d'attaque inédit.

## 🖥️ Hardware & performance

### NVIDIA — Plateforme Vera Rubin

La plateforme Vera Rubin comprend 7 puces co-conçues qui éliminent les goulots d'étranglement des déploiements multi-fournisseurs :

| Puce | Rôle |
|------|------|
| Vera CPU | Remplace le Grace CPU |
| Rubin GPU | Successeur du Blackwell |
| NVLink 6 Switch | Interconnexion intra-rack |
| ConnectX-9 SuperNIC | Réseau haute performance |
| BlueField-4 DPU | Traitement données / sécurité |
| Spectrum-6 Ethernet | Switch réseau centre de données |

Le rack NVL72 successeur atteint 260 TB/s de bande passante NVLink — NVIDIA affirme que cela dépasse la bande passante totale de l'internet. Livraison H2 2026. [(NVIDIA)](https://investor.nvidia.com/news/press-release-details/2026/NVIDIA-Kicks-Off-the-Next-Generation-of-AI-With-Rubin--Six-New-Chips-One-Incredible-AI-Supercomputer/default.aspx)

Le MI450 est conçu pour s'associer aux CPU EPYC « Venice » dans les racks IA. [(Techi.com)](https://www.techi.com/amd-meta-60b-deal-nvidia-ai-monopoly/)

### Intel — Premier support NPU complet dans MLPerf Client v0.6
